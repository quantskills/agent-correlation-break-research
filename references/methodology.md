# Methodology / 方法说明

本 Agent 用 Pandadata 的主力合约相关性证据，观察资产之间的联动关系是否发生断裂。它适合用于组合风险复盘、风格切换观察和分散假设检查。

## 核心问题

组合原有相关性和分散假设是否还可靠？

## 使用的数据

- Pandadata method: `get_future_dominant_corr`
- 主要产物：`outputs/live/correlation_matrix.csv`
- 关键图表：
  - `outputs/live/charts/correlation_heatmap.png`
  - `outputs/live/charts/cross_correlation_bars.png`
  - `outputs/live/charts/scorecard_metrics.png`

## 判断逻辑

1. 读取主力合约之间的相关性矩阵。
2. 把同品种自相关和跨品种相关性分开处理。
3. 观察最大绝对相关、最小绝对相关和相关性跨度。
4. 当相关性跨度扩大，说明组合内部可能出现强弱联动分化。
5. 当原本应当分散的品种同步走强或同步走弱，提示分散假设可能失效。

## 输出解释

- `相关性断裂观察`：跨品种相关性差异明显，需要复核组合风险假设。
- `相关性正常观察`：当前证据不足以说明相关性结构发生明显变化。
- `相关性跨度`：强相关和弱相关之间的差距，越大越说明结构分化明显。

## 适合继续追问的问题

- 哪些资产组正在从低相关变成高相关？
- 哪些原本联动的资产正在脱钩？
- 当前组合的风险预算是否还使用旧相关性矩阵？
- 相关性变化是趋势切换，还是短窗口噪声？

## 使用边界

这个 Agent 只提供研究和风险观察，不预测单个品种方向，也不替代组合层面的完整压力测试。
